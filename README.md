# 🌸 Generation Future Hub

### Premium Japanese Language Training & Visa Consultancy
**Building Bridges Between Nepal & Japan**

Generation Future Hub is a high-end educational consultancy based in Gorkha, Nepal. This landing page is designed with an "Apple-level" aesthetic, combining minimalist Japanese design with high-conversion motion graphics.

---

## ✨ Premium Features

### 🎨 Visual Excellence
- **Sakura Motion Engine**: Custom-built canvas animation simulating falling cherry blossoms with natural drift and swing physics.
- **Glassmorphism UI**: Frosted glass navigation and cards with real-time backdrop-blur effects.
- **Dynamic Micro-interactions**: 
  - **3D Tilt Cards**: Course and service cards rotate in 3D relative to cursor position.
  - **Magnetic Buttons**: Call-to-action buttons subtly attract toward the cursor.
  - **Custom Interactive Cursor**: A smooth-follwing ring cursor that scales and glows on interactive elements.

### 🚀 Performance & Motion
- **Intersection Animation System**: Scroll-triggered reveals for all sections using `IntersectionObserver`.
- **Eased UI Counters**: Progress metrics count up with organic quint-easing curves.
- **Parallax Layers**: Layered depth effects with floating badges and background orbs.
- **Throttled Scroll Handling**: High-performance scroll listeners optimized with `requestAnimationFrame`.

### 📱 Responsive & Professional
- **Modern Grid Layout**: Fluid transitions between desktop, tablet, and mobile.
- **Professional Typography**: Curated font hierarchy using *Outfit* and *Inter*.
- **Japan-Inspired Palette**: Elegant use of Crimson, Sakura Pink, and Gold accents on a clean Ink/White base.
